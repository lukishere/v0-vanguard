# Logos de Experiencia

Esta carpeta debe contener los siguientes archivos SVG de los logos de las empresas:

1. **ibm.svg** - Logo de IBM
2. **hp.svg** - Logo de HP
3. **ey.svg** - Logo de EY (Ernst & Young)
4. **logicalis.svg** - Logo de Logicalis
5. **microsoft.svg** - Logo de Microsoft

## Fuentes recomendadas para descargar los logos:

- **IBM**: https://www.ibm.com/brand/systems/developer/developer-day-brand-assets
- **HP**: https://www8.hp.com/us/en/hp-information/brand-assets.html
- **EY**: https://www.ey.com/en_gl/brand-assets
- **Logicalis**: https://www.logicalis.com (sitio oficial)
- **Microsoft**: https://www.microsoft.com/en-us/legal/intellectualproperty/trademarks/usage/logo

## Notas:

- Los logos deben estar en formato SVG para mejor calidad
- Si no encuentras SVG, puedes usar PNG pero cambia la extensión en `lib/content/about.ts`
- Los logos se mostrarán en escala de grises por defecto y en color al pasar el cursor
